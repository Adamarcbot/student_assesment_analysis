Student Assessment Analysis System

This project helps an academic department organize and analyze
internal assessment marks of students across multiple subjects.

Marks are usually stored without analysis, which makes it hard
for faculty to identify students who need attention or improvement.

process steps:
- Stores student marks in a structured way
- Calculates basic performance indicators
- Highlights students who may need improvement

Faculty members can get:
- Low-performing students
- Subject-wise performance patterns
- Students who need academic support
